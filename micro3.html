Introduction

The API gives geolocation of any area by giving longitude and latitude of area
The API is live on deployment server i.e. Deta.sh : https://6plwl4.deta.dev//

Installation Locally

In order to run the API locally clone the repository from my github account : https://github.com/Ani-Bos/Microtask3.git

Then cd third_micro

Then install the dependencies as required for this microtask like cors axios and express like npm i express cors axios nodemon.

Then run " npm run dev "

In order to fetch the API and to get a response from get request use a query i.e. "api/getAdd?lat=[latitude]&lon=[longitude]"

About repository

This repository is created for building an API for MicroTask 3 given by HostelDevta in developement phase.I have created an express server and by using get request I am fetching an api from https://nominatim.org/release-docs/develop/api/Reverse/ and using this link I have created a folder API under which I have created a file "geolocation.js" in which i am fetching this API then exporting it and finally on index.js on using it finally displaying on port given.
