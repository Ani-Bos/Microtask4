Introduction

The API is used to get user information in a temporary database created in form of a JSON file.
The API is live on deployment server i.e. Deta.sh : https://ryc3wa.deta.dev/

Installation Locally

In order to run the API locally clone the repository from my github account : https://github.com/Ani-Bos/Microtask1.git

Then cd first_micro

Then install the dependencies as required for this microtask like cors and express like npm i express cors .

Then run " node index.js "

In order to fetch the API and to get a response from get request use "/service" path

About repository

This repository is created for building an API for MicroTask 1 given by HostelDevta in developement phase.I have created an express server and by using get request we are first reading the data.json file and finally in app we are calling data and exporting it as default app.

API Endpoints
Data in form of JSON

The schema of the model is in form of an array like this :

  • {
  • "user1": {
  • "id":1,
  • "firstname":"Munonye",
  • "lastname":"Kindson",
  • "email":"kany@gmail.com"
  • }
  • }
To get the response of API
# Method : GET
# Endpoint : '/'
# This will result in "Hello"
# Endpoint : "/service"
# This will result in "showing the {username , email , password} from the Data.json"
